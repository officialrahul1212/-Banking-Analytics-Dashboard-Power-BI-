# 📊 Banking Analytics Dashboard (Power BI)

## 🔍 Project Overview

This project presents a comprehensive **Banking Analytics Dashboard** built using Power BI.
It provides insights into customers, accounts, transactions, loans, and KYC status to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze customer and account distribution
* Track transaction and loan performance
* Monitor KYC completion and status
* Identify branch-wise and city-wise trends

---

## 📁 Dashboards Included

### 1️⃣ Transactions & Loan Analytics

* Total Transactions, Loan Amount, Payments
* Transaction type breakdown (Deposit vs Withdrawal)
* Loan distribution by type (Home, Personal, Auto)
* Loan status (Active vs Closed)

---

### 2️⃣ Banking Overview Dashboard

* Total Customers, Accounts, Loans, Balance
* Branch-wise account distribution
* Loan distribution insights
* City-level filtering for deeper analysis

---

### 3️⃣ Customer Insights Dashboard

* Total Customers and KYC metrics
* KYC Completion % and Verification
* Customers by City
* KYC Status (Completed, Pending, Under Review)

---

### 4️⃣ Accounts Performance Dashboard

* Total Accounts and Balance metrics
* Account type distribution (Savings vs Current)
* Branch performance analysis
* Balance distribution across branches

---

## 📊 Key Insights

* Loan distribution is fairly balanced across categories
* Majority of accounts are Savings accounts
* Some branches show significantly higher performance
* KYC completion rate indicates scope for improvement
